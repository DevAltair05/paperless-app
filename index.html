Step 1 — Image upload to Firebase Storage
User taps "Scan" on mobile. The camera captures the receipt and uploads to receipts/{userId}/{timestamp}.jpg. A Cloud Function triggers on the new file.
Step 2 — AI API call (GPT-4o mini)
Function fetches a signed URL and sends it to the AI with a structured extraction prompt. The model reads even skewed or low-light images.
"system": "Extract receipt data. Return ONLY JSON: merchant, date, total, tax, category."
Step 3 — Parsed JSON → Firestore
AI returns a clean object; the function validates fields and writes to receipts/{receiptId}.
{ "merchant": "Shake Shack", "total": 42.80, "category": "Food" }
Step 4 — Firestore schema
Each receipt saved under receipts/{receiptId}. The users/{userId} doc tracks scanCount for free-tier enforcement.
users/{userId}: { plan, scanCount, email }
receipts/{receiptId}: { userId, imageUrl, merchantName, amount, date, tax, category, createdAt }

About the developer

Dev Nahin built Paperless to solve a real pain point: freelancers and small business owners spending hours manually entering receipts into spreadsheets at tax time. The app uses AI to turn a photo into a clean expense record in under three seconds.

Paperless is an indie Micro-SaaS product — designed, engineered, and shipped solo, from backend Cloud Functions to mobile UI.
