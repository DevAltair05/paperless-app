Step 1 — Upload to Firebase Storage
Camera captures the receipt → uploaded to receipts/{userId}/{timestamp}.jpg → Cloud Function triggers automatically.
Step 2 — GPT-4o mini AI call
Signed URL sent to AI with secret extraction prompt. Reads skewed, crumpled, or low-light photos.
"system": "Extract receipt data. Return ONLY JSON."
Step 3 — Parsed JSON → Firestore
AI returns clean object → validated → written to receipts/{receiptId}.
{ "merchant": "Shake Shack", "total": 42.80, "category": "Food" }
Step 4 — Firestore schema
Each receipt saved under receipts/{receiptId}. User doc tracks scanCount for free-tier enforcement.
users/{userId}: { plan, scanCount, email }
receipts/{id}: { userId, imageUrl, merchantName, amount, date, tax, category, createdAt }
